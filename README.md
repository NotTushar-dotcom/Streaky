# Streaky

Streaky is a colorful, gamified streak-tracking mobile app focused on helping users stay consistent with coding, gym, learning, content creation, and self-improvement habits.

## Vision

Build a dopamine-driven consistency platform that feels playful, motivating, and emotionally rewarding instead of boring like traditional productivity apps.

## Tech Stack

* React Native
* Expo
* TypeScript
* Expo Router
* Claude Opus for assisted development

## Current Status

🚧 In active development — currently building the Today Dashboard experience.
